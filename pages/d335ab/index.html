    复习MySql数据库1

    # 数据库

    MySQL

    • MySQL默认存储引擎为InnoDB

    • 不要在数据库存储文件（比如图片）

    • 尽可能把所有列定义为NOT NULL

    • 使用时间戳TIMESTAMP(4个字节)或者DATETIME（8个字节）存储时间

    • 财务相关的数据用decimal类型存储，每四位字节存储九位数字，小数点占用一个字节

    • 模糊查询避免使用双百分号

    • 插入的时候要指定列名

    • 如果没有重复值要用union all 而不是union

    MySQL索引

    聚簇索引

    索引结构和数据一起放的索引，找到了索引下标也就找到了数据，更新代价大

    非聚簇索引

    索引结构和数据没有一起放，非聚簇索引数据存放的是主健索引的地址，可能会二次回表，

    • 主健索引 ：加速查询 + 列值唯一 （不可以为NULL）
    • 普通索引： 加速查询
    • 唯一索引： 加速查询 + 列值唯一 （可以为NULL）
    • 覆盖索引： 索引包含或者覆盖所查询字段的值
    • 联合索引： 几列联合的索引
    • 前缀索引：只适用于字符串数据

    MySQL事务隔离级别

    隔离级别有： 读取未提交，读取已提交， 可重复度，可串行化

    • 脏读： 一个事务修改了数据但是没提交， 另一个事务读到了修改之后的数据， 然后第一个事务回滚
    • 不可重复读：一个事务读取数据，另一个事务也去读取，第一个事务修改数据， 第二个事务在读取就发生了变化
    • 幻读： 比如获取总行数, 一个事务添加数据，一个事务读取数据， 添加之后，另一个事务读取就会发生变化。

    1.力扣2446判断两个事件是否存在冲突

    题目链接:2446. 判断两个事件是否存在冲突 - 力扣（LeetCode） (opens new window)


    把时间想象成两条线段， 相交的话只有两种情况， 线段1在前， 线段2在前

    class Solution {
public:
    bool haveConflict(vector<string>& event1, vector<string>& event2) {
       string s1 = event1[0];
       string s2 = event1[1];
       string s3 = event2[0];
       string s4 = event2[1];
       if(s2 >= s3 && s2 <= s4 || s4 >= s1 && s4 <= s2){
           return true;
       }
       return false;
    }
};

    2.力扣2513最小化数组中的最大值

    题目链接:2513. 最小化两个数组中的最大值 - 力扣（LeetCode） (opens new window)


    二分来做， 通过二分答案， check函数判断当前数字能供给arr1多少个数，供给arr2多少个数,然后看他们公共的有多少个数

    class Solution {
public:
    int minimizeSet(int divisor1, int divisor2, int uniqueCnt1, int uniqueCnt2) {
        
        long long l = -1, r = 1e18 + 10;
        while(l + 1 < r){
            long long mid = (l + r) / 2;
            //[1,mid]
            //2 4 6 8 10
            long long x1 = mid / divisor1;
            long long x2 = mid / divisor2;
            long long gc = __gcd(divisor1,divisor2);
            long long gd = 1LL * divisor1 * divisor2 / gc;
            long long x3 = mid / gd;
            long long t = mid - (x1 + x2 - x3);
            int ok = 1;
            if(mid - x1 < uniqueCnt1){
                ok = 0;
            }
            if(mid - x2 < uniqueCnt2){
                ok = 0;
            }
            if(mid - x1 + mid - x2 - t < uniqueCnt1 + uniqueCnt2){
                ok = 0;
            }
            if(ok == 1){
                r = mid;
            }
            else{
                l = mid;
            }
        }
        return r;
    }
};

    3.力扣2501数组中最长的方波

    题目链接: 2446. 判断两个事件是否存在冲突 - 力扣（LeetCode） (opens new window)


    每次找到当前元素的开平方之后数出现的次数， dp[x] = dp[sqrt(x)] + 1 前提是sqrt(x)在之前出现过

    class Solution {
public:
    int longestSquareStreak(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        int dp[100010] = {0};
        int a[100010] = {0};
        //2 3 4 6 8 16
        int res = 0;
        for(int i = 0; i < nums.size(); i++){
            int x = nums[i];
            a[x]++;
            double dq = sqrt(x);
            int iq = int(dq);
            if(iq == dq){
                if(dp[iq] == 0 && a[iq] > 0){
                    dp[iq] = 1;
                }
                dp[x] = dp[iq] + 1;
            }
            res = max(res, dp[x]);
        }
        if(res == 0 || res == 1){
            return -1;
        }
        return res;

    }
};
