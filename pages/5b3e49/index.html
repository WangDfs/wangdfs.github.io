    # Servlet

    Servlet是运行在服务器上的一个java小程序，可以接受客户端发过来的请求，并响应数据给客户端

    在java中，规范就是接口，而Servlet是javaEE的一个规范，也就是接口。

    # 手动实现一个Servlet小程序

    1.先编写一个类去实现这个接口

    2.实现接口中的service方法（可以判断请求的类型），处理请求，并响应数据。

    3.去web.xml中去配置Servlet程序要访问的地址。

    配置Servlet要访问的地址时要以斜杠开头

    # http协议

    客户端和服务器端之间进行通信时，发送的数据，需要遵守的规则，叫做http协议

    客户端给服务器发送数据叫做请求

    服务器给客户端回传数据叫做响应

    请求由请求行，请求头，和请求体组成

    请求行由 请求方式 + （请求资源路径 + ？+ 请求参数） + 版本号

    请求头是由键值对组成 （key : value)

    响应状态码

    200 表示请求成功

    302 表示请求重定向

    404 表示请求服务器已经收到了，但是你要的数据不存在（请求地址错误）

    500 表示服务器已经收到请求，但是服务器内部错误（代码错误）

    # HttpServletRequest类

    通过HttpRequest类可以获取到所有浏览器发来的请求信息

    i. getRequestURI() 获取请求的资源路径

    ii. getRequestURL() 获取请求的统一资源定位符（绝对路径）

    iii. getRemoteHost() 获取客户端的 ip 地址

    iv. getHeader() 获取请求头

    v. getParameter() 获取请求的参数

    vi. getParameterValues() 获取请求的参数（多个值的时候使用）

    vii. getMethod() 获取请求的方式 GET 或 POST

    viii. setAttribute(key, value); 设置域数据

    ix. getAttribute(key); 获取域数据

    x. getRequestDispatcher() 获取请求转发对象

    base标签可以用来设置当前页面所有相对路径的问题
