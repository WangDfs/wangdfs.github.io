    字符串哈希

    提示

    1.将一段字符串以数值的方式表示

    2.可能会有冲突

    3.方便比较两段字符串是否相等

    设H(i)为str[1...i]的哈希值
H(0) = 0
有递推式H(i) = H(i - 1) × P + str[i]
str[l - r] = H(r) - H(l - r) × p ^(r-l+1)
P一般取131或者13331

    注意

    java中并没有无符号长整型

    通过一些特定的方式来实现无符号整形

    # java模板

    public class Main{
    
    public static void main(String[] args){
         for(int i = 1; i <= s.length(); i++){
            h[i] = (h[i - 1] * 131 + str.charAt(i)) & 0x0FFFF;;
        } 
        p[0] = 1;
        for(int i = 1; i <= s.length(); i++){
            p[i] = (p[i - 1] * 131) & 0x0FFFF;;
        }
    }
    
    public static long find(int l, int r){
        return 0x0FFFF & h[r] - h[l - 1] * p[r - l + 1];
    }
    
}
