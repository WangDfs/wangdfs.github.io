    Redis存储的是key-value结构的结构，其中key是字符串类型，value有5种常用的数据类型

    • 字符串string
    • 哈希hash
    • 列表list
    • 集合set
    • 有序集合sorted set

    # 哈希操作命令·
    • HSET key field value 将哈希表key中的字段field的值设为value
    • HGET key field 获取存储在哈希表中指定字段的值
    • HDEL key field 删除存储在哈希表中的指定字段
    • HKEYS key 获取哈希表中所有字段
    • HVALS key 获取哈希表中所有值
    • HGETALL key获取哈希表指定key的所有字段和值
